Для просмотра файлов необходимо использовать:
- PyCharm
- Jupyter Notebook

game-old.py - файл самой игры. Можно запускать, играть.
run_this.py - игра + прикручено обучение
logger.py - логгер, в отдельном классе. Импортируется и инициируется по названию файла в game_old
bica.py - описание модели BICA. Пока не прикручено в game_old
